创建个人博客

创建hexo的个人博客
先下载nodejs 网站地址nodejs.org（npm为包管理器）
本人cmder node -v（查看node版本）
npm install -g cnpm –registry=https://registry.npm.taobao.org
cnpm install -g hexo-cli(安装hexo的框架)
创建空文件夹
进入空文件夹
hexo init（生成博客）
hexo s（启动博客）
浏览器输入地址

hexo n “标题”
cd source/_posts/
